# SpotLine 데모 이미지 제공 요청서 (프론트엔드 준비 완료)

## ✅ 프론트엔드 준비 상태
프론트엔드에서 이미지 최적화 및 관리 시스템을 구축했습니다. 백엔드에서 실제 이미지만 제공하면 즉시 적용됩니다.

### � 구현된 기능
- **이미지 최적화**: Next.js Image 컴포넌트 기반 최적화
- **로딩 상태**: 스켈레톤 UI 및 로딩 인디케이터
- **에러 처리**: 이미지 로딩 실패 시 대체 이미지 표시
- **플레이스홀더**: 임시 Unsplash 이미지로 데모 가능
- **환경 변수**: 이미지 URL 중앙 관리

## 🖼️ 현재 상황
SpotLine 데모 시스템에서 이미지들이 임시 플레이스홀더로 표시되고 있습니다. 실제 이미지 제공 시 즉시 교체 가능한 구조로 준비되어 있습니다.

## 📋 요청 사항
백엔드에서 데모용 이미지를 실제 접근 가능한 URL로 제공해주세요.

## 🎯 필요한 이미지 목록

### 1. 메인 데모 매장 이미지
- **매장명**: 아늑한 카페 스토리

- **설명**: 따뜻하고 아늑한 카페 분위기의 이미지

#### 2-1. 달콤한 베이커리
- **파일명**: `bakery-001.jpg` (권장)
- **권장 크기**: 400x300px 이상
- **설명**: 갓 구운 빵이나 베이커리 매장 이미지

#### 2-2. 조용한 서점
- **파일명**: `bookstore-001.jpg` (권장)
- **권장 크기**: 400x300px 이상
- **설명**: 책이 진열된 서점 내부나 독서 공간 이미지

#### 2-3. 꽃향기 플라워샵

- **권장 크기**: 400x300px 이상
- **설명**: 다양한 꽃들이 진열된 플라워샵 이미지

#### 2-4. 작은 갤러리
- **파일명**: `art-001.jpg` (권장)
- **권장 크기**: 400x300px 이상
- **설명**: 작품이 전시된 갤러리나 아트 공간 이미지

## 🔧 구현 방법 (3가지 옵션)

### ✅ 옵션 A: 정적 파일 제공 (가장 간단, 권장)
```bash
# 프로젝트 루트의 public 폴더에 이미지 배치
/public/demo/
├── cafe-001.jpg
├── bakery-001.jpg
├── bookstore-001.jpg
├── flower-001.jpg
└── art-001.jpg
```

**설정 방법:**
1. 이미지 파일들을 `/public/demo/` 폴더에 배치
2. `.env.local` 파일에서 다음 설정:
```env
NEXT_PUBLIC_DEMO_IMAGE_BASE_URL=/demo
```

### 옵션 B: CDN URL 제공
```env
NEXT_PUBLIC_DEMO_IMAGE_BASE_URL=https://cdn.spotline.com/demo
```

### 옵션 C: API 엔드포인트로 제공
```env
NEXT_PUBLIC_DEMO_IMAGE_BASE_URL=https://api.spotline.com/images/demo
```

- **성능 최적화**: Next.js Image 컴포넌트 사용
- **반응형**: 다양한 화면 크기 지원

### 3. 현재 임시 이미지

- 베이커리: `https://images.unsplash.com/400x300/?bakery,bread,pastry`

## 🎨 이미지 스타일 가이드

### 1. 전체적인 톤앤매너
- **분위기**: 따뜻하고 친근한 느낌
- **색감**: 자연스럽고 부드러운 톤
- **품질**: 고품질, 전문적인 사진
- **해상도**: 선명하고 깨끗한 이미지

### 2. 매장별 이미지 가이드

#### 카페 (메인 매장) - 우선순위 1
- 따뜻한 조명의 카페 내부
- 커피와 디저트가 보이는 테이블
- 아늑하고 편안한 분위기
- 사람들이 편안하게 있는 모습 (선택사항)

#### 베이커리 - 우선순위 2
- 갓 구운 빵들이 진열된 모습
- 따뜻한 베이커리 분위기
- 빵의 질감이 잘 보이는 이미지
- 다양한 종류의 빵과 페이스트리

#### 서점 - 우선순위 3
- 책이 정돈되어 진열된 서가
- 독서 공간이나 카페 공간
- 조용하고 아늑한 분위기
- 책을 읽는 사람들 (선택사항)

#### 플라워샵 - 우선순위 4
- 다양한 색깔의 꽃들
- 정돈된 꽃 진열
- 밝고 생기있는 분위기
- 꽃다발이나 화분들

#### 갤러리 - 우선순위 5
- 작품이 전시된 벽면
- 깔끔하고 모던한 전시 공간
- 예술적인 분위기
- 관람객들이 작품을 감상하는 모습 (선택사항)
